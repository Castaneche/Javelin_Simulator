Javelin_Simulator version : 1.0
This Software has been developed in C++
Libraries used : 
	- SFML (http://www.sfml-dev.org/index.php)
	- ImGui (https://github.com/ocornut/imgui)
Equation :
	y = -(g/(2v0^2(cosa)^2))x^2 + (sina/cosa)x + h0
	v0 = initial velocity (m/s)
	a = angle (rad)
	h0 = initial height (m)
